Data Analysis Short Course
Stack Overflow 2024 Developer Survey Analysis
Project Overview

This project analyzes data from the 2024 Stack Overflow Developer Survey to identify trends in programming languages, databases, cloud platforms, web development technologies, developer demographics, and job satisfaction. The analysis was performed using Python, SQLite, SQL queries, and data visualization tools to transform raw survey data into meaningful insights.

The project was completed as part of the IBM Data Analyst Professional Certificate Capstone Project.

Objectives

The main objectives of this project were to:

Analyze current technology usage among developers.
Identify future technology trends and developer preferences.
Explore relationships between technologies and job satisfaction.
Examine demographic characteristics of the global developer community.
Create visualizations and dashboards to communicate findings effectively.

Key Findings
Programming Languages
JavaScript is the most widely used programming language.
Python is the most desired programming language for future use.
JavaScript is the most admired programming language.
TypeScript users reported the highest job satisfaction.
Databases
PostgreSQL is both the most used and most desired database.
Microsoft SQL Server users reported the highest job satisfaction.
Cloud Platforms
Amazon Web Services (AWS) is the most used and most desired cloud platform.
Microsoft Azure is the second most popular cloud platform.
Web Development
React is the most preferred web framework (29%).
Node.js follows closely (27.7%).
ASP.NET Core users reported the highest job satisfaction.
Developer Demographics
The United States has the highest number of survey respondents.
Bachelor's and Master's degree holders make up the majority of respondents.
Full-Stack Developer is the most common developer role.
Development Environment
Windows is the most widely used operating system.
Visual Studio Code is the most popular development and collaboration tool.
Artificial Intelligence
Writing code is the primary use case for AI-assisted development tools.
Embedded Technologies
Raspberry Pi (31.2%) and Arduino (20.6%) are the most desired embedded platforms.

Conclusion

The analysis reveals continued growth in cloud computing, modern web development, AI-assisted programming, and open-source technologies. AWS, PostgreSQL, React, and Python remain highly influential technologies, while developer satisfaction strongly aligns with technologies that provide productivity, flexibility, and strong community support.

These insights can help developers, employers, educators, and organizations make informed decisions regarding skill development, technology adoption, and workforce planning.
